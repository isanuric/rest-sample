## rest-sample

A simple REST API with spring boot.


### Quick Start
`$ git clone git@github.com:isanuric/rest-sample.git
`

`$ cd rest-sample
`

`$ mvn clean install
`

run integration tests.

or: 

`$ mvn spring-boot:run
`

On your browser:

`localhost/8080
`

`localhost/8080/users
`

`localhost/8080/user/1
`
